# Context
- Uses latest Laravel (v8)

# Prerequisites
- PHP ^7.3 or later.
- Composer

# Setup
- go to ./src
- composer install
- npm install
- php artisan migrate

# Run
- go to ./src
- php artisan serve
- npm run dev

# Testing
- go to ./src
- vendor/bin/phpunit

# Usage
- You may use the test csv file in ./test-csvs
